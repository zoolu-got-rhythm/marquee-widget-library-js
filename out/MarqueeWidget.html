Class: MarqueeWidget

MarqueeWidget(maxWidth, title, delayInMs, fontSettingsString, pixelJump)

new MarqueeWidget(maxWidth, title, delayInMs, fontSettingsString, pixelJump)

Represents a marquee widget.
Parameters:
Name Type Description
maxWidth number width of text container must be smaller than width of contained text.
title string text string you want to display in the widget.
delayInMs number speed in millasecs that text is moved to left by x pixels (pixelJump).
fontSettingsString string font settings: e.g: "bold 28px arial".
pixelJump number number of pixels to move to the left by each step, (default value is 1px).
